# VZ Downloader - Seu Baixador de videos do youtube

O VZDownloader é uma alternativa para que você possa baixar seus vídeos do youtube facilmente completamente sem anúncios! 
Nosso objetivop é democratizar o baixamento de vídeos para produtores de vídeo e até você, qualquer um, de graça!
A VZ Team não se responsabiliza a questão de copyright, somos apenas um baixador de vídeos, e a pirataria e cópias de artistas licenciados é de total responsabilidade do usuário, por isso, use com cautela!
